LPL for WordPress Loader Plus LightBox using various options in admin panel

== Description ==

This plugin provides an easy way to before all body 'Onload/Bodyload/Windowload/Pageload' first loading show animation than lightbox show. 
i.e. "Please Wait, Loading…"  animation image show for DOM (Document Object Model) or full body page/document load than show lightbox after show body content.

Up coming version will have many more options

== Installation ==

1. Upload `lpl.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto: Admin Dashboard > LPL Load admin menu to configure the plugin settings
